#  Insurance Purchase Prediction

This project predicts whether a person will buy insurance based on their **age** and **affordability** using a Machine Learning model.

The model is trained using **Logistic Regression (SGDClassifier)** and deployed using **Streamlit**.

---

##  Overview

- Input:
  - Age
  - Affordability (0 = No, 1 = Yes)
- Output:
  - Will Buy Insurance (Yes / No)

---

##  Model Details

- Algorithm: Logistic Regression (SGDClassifier)
- Type: Binary Classification
- Loss Function: Log Loss
- Optimization: Stochastic Gradient Descent

---

##  Tech Stack

- Python  
- NumPy  
- Pandas  
- Scikit-learn  
- Streamlit
